Membuat web e-commerce dengan PHP dan MySQL

06.49  Kuliah  No comments



Posted on 20 September 2010 by Kang Agus
 
 
 
 
 
 
1 Vote


Pada artikel kali ini, saya akan sharing tentang contoh web ecommers sederhana dengan PHP. Aplikasi ini hasil dari tugas praktikum pemrograman web di kampus saya. pada contoh web sederhana ini saya menggunakan database untuk pembuatan fitur shopping chart.
untuk referensi silahkan baca artikel ini :
http://ilmukomputer.org/2008/06/01/membuat-shopping-cart-sederhana-menggunakan-php/

gunakan username dan password : agus
CATATAN
Bagi teman-teman yang gagal masuk halaman admin, setelah saya lihat kembali script programnya dan saya jalankan pada xampp versi 1.6.  ternyata bisa berjalan dengan baik. mungkin masalahnya ada pada versi PHP yang digunakan saat aplikasi tokoonline ini dibuat yaitu masih PHP versi 4. dimana salah satu perbedaan mendasarnya adalah pada tag pembuka PHP yang mengalami perubahan dari <? (php 4) menjadi <?php (php 5). sehingga solusinya ganti (find and replace) semua tag lama <? menjadi tag baru <?php
CMIIWJika ada teman-teman yang lebih tau solusinya silahkan dikoreksi letak kesalahannya. thanks B4. dan aplikasi yang saya buat mungkin tidak terlalu bagus dari segi design sistemnya. silahkan di pelajari dan semoga dapat di kembangkan
Sekian artikel dari saya, semoga bermanfaat untuk temen-temen dan anda semua yang sedang mempelajari PHP.
